# 拡張機能 更新・確認フロー（鉄の掟）

AI（私）とユーザー（あなた）の間で、以下の手順を徹底します。

## 1. AI側の責務：自動更新とビルド
AIはコード編集後、必ず以下のスクリプトを実行して「更新パッケージ」を作成します。

```powershell
.\scripts\update_extension.ps1
```

**このスクリプトが行うこと:**
1. `manifest.json` のバージョンを自動で上げる（例: 1.0.0 → 1.0.1）
2. `App.tsx` 内のバージョン表示を書き換える
3. `npm run build` を実行して、確実に最新コードを反映する
4. 完了音を鳴らす 🔔

## 2. ユーザー側の責務：読み込み確認
スクリプト完了の通知（音）が聞こえたら、ユーザーは以下を行います。

1. Chromeの拡張機能管理ページ (`chrome://extensions`) を開く
2. **「更新」ボタンを押す**（またはGORewriteをOFF→ON）
3. GORewriteを開き、**画面左上のバージョン番号が上がっているか確認する**

## 合言葉
「ビルド完了。バージョン **vX.X.X** になりました。更新ボタンをお願いします！」
